activate_local_smoothing(diameter=None, smooth_factor=1.0, target_entity=None)[source]¶

This operation will smooth entities interactively by re-positioning the triangle nodes & triangle edges.

Parameters:
  • diameter (float, optional) – The diameter size for smoothing. If no diameter is provided, the diameter is determined by the camera position.
  • smooth_factor (float, optional) – Strength of the smoothing.
  • target_entity (trimatic.Surface | trimatic.Part, optional) – Part/surface to be smoothed. If no target entity is provided, all visible entities within the smoothing diameter are smoothed.
Raises:

RuntimeError, ValueError
Example: 
import trimatic

cylinder = trimatic.create_cylinder_part( point_1 = (0,0,0), point_2 = (0,0,100), radius = 10)
trimatic.activate_local_smoothing(2.0, 0.1, cylinder)

activate_translate_rotate(main_entity=None, move_along_entities=None, translation_step=None, rotation_step=None)[source]¶

Translate and (or) rotate objects.

Parameters:
  • main_entity (trimatic.Object) – Entity to be transformed.
  • move_along_entities (trimatic.Object | [trimatic.Object] | (trimatic.Object)) – List of entities to move along.
  • translation_step (float, optional) – The translation will be done in steps of this distance.
  • rotation_step (float, optional) – The rotation will be done in steps of this angle.
Raises:

RuntimeError, ValueError
Example:

cylinder = trimatic.create_cylinder_part( point_1 = (0,0,0), point_2 = (0,0,100), radius = 10)
print('Use the mouse to select and reposition objects. Press ESC when done.')
trimatic.activate_translate_rotate(cylinder, sphere)

print('Thank you for clicking.')

Trims off excess material of parts.

indicate_coordinate()[source]¶

Allows users to create a point in the 3D area. Filter options in the work area can be used for specific point selection.

    try:

        print('indiated coordinates are:' + str(coords))
    except:
        break

message_box(message, title, with_cancel=True)[source]¶

Pops up a non-blocking (modeless) messagebox that pauzes the script and gives control back to 3-matic. After the user presses OK or Cancel, the script will continue.

Parameters:
  • message (string) – Message text in the box.
  • title (string) – Title for the messagebox.
  • with_cancel (bool, optional) – When true, the message box will show OK. When False, the messagebox will show cancel.
Returns:

True when user pressed OK, False when user pressed Cancel.
Return type:

bool
Example: 
import trimatic

msg = "This is an example message."
title = "Example title"
trimatic.message_box(message=msg, title=title, with_cancel=True)

Sets the selection in the object tree to the specified entities.

print('Notice that surface with name ' + selected.name +  ' is selected.')
