ssm¶

registration_with_scaling(*args, fixed_entity, moving_entity, move_along_entities=(), sample_points=15.0, number_of_iterations=10)[source]¶

An iterative closest point registration with isotropic scaling.

Parameters:
  • fixed_entity (trimatic.Part) – Reference entity to be aligned with.
  • moving_entity (trimatic.Part) – Entity to align/move.
  • move_along_entities (trimatic.Object | [trimatic.Object], optional) – Either a single object or heterogeneous list of root level entities (parts, analytical primitives) that will be aligned along with the moving entity.
  • sample_points (float, optional) – The size of the point sample in percentage. A higher value implies more points and thus a higher computation time.
  • number_of_iterations (int, optional) – The number of iterations. A higher value results in better quality but increases the computation time.
Returns:

Root mean squared error of deviation between the moving entity and the fixed entity. None if operation failed.
Return type:

float
Raises:

RuntimeError, ValueError
Example: 
import trimatic
import os

application_path = os.path.dirname(trimatic.get_application_path())
trimatic.open_project(os.path.join(application_path, "DemoFiles", "SSM femur.mxp"))
original_parts = trimatic.find_group(name="Original parts").items[0:3]
[fix_entity, mov_entity] = trimatic.find_group(name="Original parts").items[3:5]
for i in original_parts: i.visible=False
trimatic.registration_with_scaling(fixed_entity=fix_entity, moving_entity=mov_entity)
warp_part(*args, fixed_entity, moving_entity, fixed_landmarks=None, moving_landmarks=None, number_of_smooth_iteration=50, bidirectional_point_correspondence=True, bad_contour_correspondence=False, triangle_correspondence=True, max_stiffness=30.0, min_stiffness=5.0, max_geometrical_error=0.01, max_non_smooth_iterations=300, subsample_ratio=0.5, convergence_tolerance=0.001, normal_constraint_angle=30.0)[source]¶

Adapts the mesh of the moving entity to create point correspondence with the fixed entity.

Parameters:
  • fixed_entity (trimatic.Part) – Fixed/target entity to be warped with.
  • moving_entity (trimatic.Part) – Moving/master entity to warp.
  • fixed_landmarks ([(float,float,float)], optional) – Landmarks on fixed/target entity.
  • moving_landmarks ([(float,float,float)], optional) – Landmarks on moving/master entity.
  • number_of_smooth_iteration (int, optional) – Number of smooth warping iterations upfront.
  • bidirectional_point_correspondence (bool, optional) – When True, search for corresponding points from the moving entity to the fixed entity and from the fixed entity to the moving entity.
  • triangle_correspondence (bool, optional) – When True, the triangle centers are included in the corresponding point search.
  • bad_contour_correspondence (bool, optional) – When True, the bad contours of the moving and fixed entities will be matched.
  • normal_constraint_angle (float, optional) – The angle [degrees] with which the normal of a corresponding point may deviate.
  • subsample_ratio (float, optional) – The fraction of points on the moving surface which search for a corresponding point in each iteration.
  • max_stiffness (float, optional) – The maximum stiffness determines the maximum strength of the connections between neighbouring points.
  • min_stiffness (float, optional) – The minimum stiffness determines the minimum strength of the connections between neighbouring points.
  • max_non_smooth_iterations (int, optional) – The maximum number of iterations after which the algorithm will stop.
  • max_geometrical_error (float, optional) – The maximum geometrical error for the Housdorff distance between the moving surface and the fixed surface.
  • convergence_tolerance (float, optional) – The tolerance for the difference between the Housdorff distance in interation n and iteration n-1 of the warping algorithm.
Returns:

Created part from the warping operation.
Return type:

trimatic.Part | (trimatic.Part)
Raises:

ValueError, RuntimeError
Example: 
import trimatic
import os

application_path = os.path.dirname(trimatic.get_application_path())
trimatic.open_project(os.path.join(application_path, "DemoFiles", "SSM femur.mxp"))
original_parts = trimatic.find_group(name="Original parts").items[0:3]
[fix_entity, mov_entity] = trimatic.find_group(name="Original parts").items[3:5]
for i in original_parts: i.visible=False
trimatic.warp_part(fixed_entity=fix_entity, moving_entity=mov_entity)
