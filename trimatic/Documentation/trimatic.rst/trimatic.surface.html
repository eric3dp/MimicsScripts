surface¶

automatic_rectangular_patching(entities, perform_preprocessing=True, low_curvature=True, part_complexity=None, allow_t_junctions=True)[source]¶

The patching operation allows preparing a part before applying the export_iges or export_step operation.

Parameters:
  • entities (trimatic.Part | [trimatic.Part]) – Part or list of parts to export.
  • perform_preprocessing (bool, optional) – Destination path to save the file(s).
  • low_curvature (bool, optional) – Set to True when the part contains a low curvature. Set to False when the part contains a high curvature.
  • part_complexity (int, optional) – Value between 1 and 10 to indicate the complexity of a part. The complexity of a part determines the patch density. Higher density will result in a more accurate representation, but more data. When specifiying None, 3-matic will automatically estimate the complexity of the selected part(s) and use the average complexity.
  • allow_t_junctions (bool, optional) – When True, the edge of one patch can be built out of 2 edges of other patches.
Returns:

For every input part: The resulting surfaces grouped into surface sets. When no patching problems have occurred, only one surface set is returned. When patching problems have occured, more than one surface set will be returned. The first will contain the successful surfaces, the other ones contain problem surfaces.
Return type:

((trimatic.SurfaceSet))
Raises:

ValueError, RuntimeError
Example: 
import trimatic

prim1=trimatic.create_cylinder_part(point_1=(0,0,0), point_2=(0,10,0), radius=5)
trimatic.surface.automatic_rectangular_patching (prim1, perform_preprocessing = True, low_curvature = True, part_complexity = None, allow_t_junctions = True)
estimate_part_complexity(entity)[source]¶

Estimates the part complexity.

Parameters:entity (trimatic.Part) – Part to estimate.
Returns:Value between 1 and 10 to indicate the complexity of a part.
Return type:int
Raises:ValueError, RuntimeError
Example: 
import trimatic

point_1=(0,0,0)
point_2=(0,10,0)
prim1=trimatic.create_cylinder_part(point_1, point_2, radius=50)
trimatic.surface.automatic_rectangular_patching (prim1, perform_preprocessing = True, low_curvature = True, part_complexity = None, allow_t_junctions = True)
Complexity1=trimatic.surface.estimate_part_complexity (prim1)
print(Complexity1)
unfold_surface(entities, seams=None, preserve_surface_connections=False)[source]¶

Performs unfold surface on selected entitites.

Parameters:
  • entities (trimatic.Surface | trimatic.SurfaceSet | [trimatic.Surface | trimatic.SurfaceSet]) – Surface(s) to unfold.
  • seams (trimatic.Curve | [trimatic.Curve], optional) – One or multiple attached curve(s) to be used as relaxer cuts in the unfolding.
  • preserve_surface_connections (bool, optional) – When True, all the existing connections between the selected entities will be preserved.
Return type:

trimatic.Part | (trimatic.Part)
Raises:

ValueError, RuntimeError
Example: 
import trimatic

sphere = trimatic.create_sphere_part(point_center=(0,0,0), radius=10)
trimatic.automatic_rectangular_patching(entities=sphere)
sphere.get_surfaces()[0].color = (0.2, 0.5, 0.8)
unfolded_surface = trimatic.unfold_surface(sphere.get_surfaces()[0])
unfolded_surface.color = (0.2, 0.5, 0.8)
