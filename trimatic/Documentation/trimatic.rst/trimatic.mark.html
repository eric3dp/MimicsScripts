mark_entities(entities, mark=True)[source]¶

The entities to mark or unmark.

Parameters:
  • entities (trimatic.Part | trimatic.Surface | trimatic.SurfaceSet | trimatic.SurfaceBorder | trimatic.SurfaceContour | trimatic.Curve | [trimatic.Part | trimatic.Surface | trimatic.SurfaceSet | trimatic.SurfaceBorder | trimatic.SurfaceContour | trimatic.Curve]) – Either a single object or a heterogeneous list of objects.
  • mark (bool, optional) – The entities will be marked (True) or unmarked (False).
Returns:

Marked triangles for every part that was processed (complete or entity in it).
Return type:

trimatic.MarkedTriangles | (trimatic.MarkedTriangles)
Raises:

RuntimeError, ValueError
Example: 
import trimatic

c = (0,0,0)
x = 3
y = 90
z = 35.4
box = trimatic.create_box_part(c,x,y,z)

trimatic.adaptive_remesh(entities=box,target_triangle_edge_length=1)

right_surf = box.find_surface("Right")
border = right_surf.get_border()

marked = trimatic.mark_entities(entities=border, mark=True)
