align_uv_automatic(uv_map, size_u=None, size_v=None)[source]¶

Align the UV region by four contour points automatically.

Parameters:
  • uv_map (trimatic.UVMap) – A UV Map with UV Regions to be aligned.
  • size_u ((float, optional)) – The length of the U axis of the aligned region in UV units. If both size_u and size_v are set to None, the aligned region will result in a square region.
  • size_v ((float, optional)) – The length of the V axis of the aligned region in UV units. If both size_u and size_v are set to None, the aligned region will result in a square region.
Returns:

UV Map
Return type:

trimatic.UVMap
Raises:

ValueError | Runtime Error (When only one of the two parameters size_u and size_v is None)
Example: 
import trimatic
import os

trimatic.open_project(os.path.dirname(trimatic.get_application_path()) + r'\DemoFiles\droid.mxp')
part = trimatic.find_part('droid')
target_surface = part.find_surface('Surface 691')
target_uv_map = trimatic.create_uv_maps(entities= target_surface)

trimatic.align_uv_automatic(target_uv_map, size_u= 1.0, size_v= 0.7)
create_uv_maps(*args, entities, seams=None, single_uv_map=True, preserve_surface_connections=False, xyz_to_u_factor=10.0, xyz_to_v_factor=10.0)[source]¶

Operation to create UV maps with surfaces/surface sets input.

Parameters:
  • entities (trimatic.Surface | trimatic.SurfaceSet | [trimatic.Surface | trimatic.SurfaceSet]) – Entities to be parameterized. Heterogeneous list of entities are allowed.
  • seams (trimatic.Curve | [trimatic.Curve], optional) – One or multiple attached curve(s) to be used as relaxer cuts in the parameterization.
  • single_uv_map (bool, optional) – When True, only one UV Map will be generated in a single parent part.
  • preserve_surface_connections (bool, optional) – When True, all the existing connections between the selected entities will be preserved in the output map.
  • xyz_to_u_factor (float, optional) – The XYZ to U factor of the initial parameterization.
  • xyz_to_v_factor (float, optional) – The XYZ to V factor of the initial parameterization.
Returns:

Newly created UV Maps.
Return type:

trimatic.UVMap | (trimatic.UVMap)
Raises:

ValueError | RuntimeError
Example: 
import trimatic

part = trimatic.create_box_part([0.0, 0.0, 0.0], 10.0, 10.0, 10.0)
front_surface = part.find_surface("Front")
left_surface = part.find_surface("Left")

trimatic.create_uv_maps(entities= (front_surface, left_surface), preserve_surface_connections= True)
optimize_uv(entities, iterations=100, method=<OptimizeMethod.AnglesAndArea: 'Angles and Area'>, preserve_uv_border=True)[source]¶

Optimizes the UV Parameterization based on the selected method for the number of iterations set by the user.

Parameters:
  • entities (trimatic.Part | trimatic.UVRegion | trimatic.UVMap | [trimatic.Part | trimatic.UVRegion | trimatic.UVMap]) – Parameterized entities to be optimized. Heterogeneous list of entities are allowed.
  • iterations (int, optional) – Number of iterations to perform the operation before termination.
  • method (trimatic.OptimizeMethod, optional) – The method to be used to optimize the UV parameterizations.
  • preserve_uv_border (bool, optional) – If set to True, the UV border will not be changed after the optimization.
Returns:

One or more optimized UV Maps.
Return type:

trimatic.UVMap | (trimatic.UVMap)
Raises:

ValueError | RuntimeError
Example: 
import trimatic
import os

trimatic.open_project(os.path.dirname(trimatic.get_application_path()) + r"\DemoFiles\Cup.mxp")
target_part = trimatic.find_part("Lattice Volume")
target_surface = target_part.find_surface_set("Subtraction result-001-001+-001-001-2+")
trimatic.create_uv_maps(entities = target_surface, preserve_surface_connections = True)

optimized_entity = trimatic.optimize_uv(entities = target_part, iterations = 100, method = trimatic.OptimizeMethod.Area, preserve_uv_border = True)
