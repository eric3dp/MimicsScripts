collision_detection(entities, clearance_check=0.0)[source]¶

Detect collision between parts by considering acceptable clearance between parts.

Parameters:
  • entities ([trimatic.Part] | (trimatic.Part)) – A list of parts to check whether the parts are colliding.
  • clearance_check (float) – The minimum distance between parts that can be considered collision. If the distance between parts is less than or equal to this value, the parts are considered to be colliding.
Returns:

A tuple of tuples containing colliding parts as pairs.
Return type:

((trimatic.Part, trimatic.Part))
Raises:

RuntimeError, ValueError
Example: 
import trimatic

cylinder_1 = {'pnt1':(0,0,0), 'pnt2':(0,0,100),'radius':100}
cylinder_2 = {'pnt1':(50,0,0), 'pnt2':(50,0,100),'radius':100}
cyl1 = trimatic.create_cylinder_part(cylinder_1['pnt1'], cylinder_1['pnt2'], cylinder_1['radius']) 
cyl2 = trimatic.create_cylinder_part(cylinder_2['pnt1'], cylinder_2['pnt2'], cylinder_2['radius'])
detect = trimatic.collision_detection(entities=[cyl1, cyl2], clearance_check=0.0)
if detect:
    print('The following parts are colliding: ') 
    print(detect)
else:
    print("No collision found.")
