create_angle_measurement_3points(center, point1, point2, measurement_coordinates=<MeasurementCoordinates.XYZ: 'XYZ'>, user_defined_cs=None)[source]¶

Creates an angle measurement between 3 points. The position of the angle measurement is defined by the sum of the center-point1 vector and the center-point2 vector.

Parameters:
  • center ([float] | (float) | trimatic.Point) – Center point of the measurement.
  • point1 ([float] | (float) | trimatic.Point) – Point1 of the measurement.
  • point2 ([float] | (float) | trimatic.Point) – Point2 of the measurement.
  • measurement_coordinates (trimatic.MeasurementCoordinates, optional) – Choose to measure the angle in all 3 axes or only one plane.
  • user_defined_cs (trimatic.CoordinateSystem, optional) – Single entity of trimatic.CoordinateSystem. If this parameter is not valid, then the world coordinate system will be applied. The measurement will be taken based on the coordinate system of the selected entity.
Returns:

Created measurement.
Return type:

trimatic.AngleMeasurement
Raises:

RuntimeError, ValueError
Example: 
import trimatic

p1 = (0,0,0)
p2 = (21,4.3,67)
p3 = (1,68,-1000)
trimatic.create_angle_measurement_3points(p2,p1,p3,trimatic.MeasurementCoordinates.XZ)
create_angle_measurement_line_to_line(line_from, line_to, measurement_coordinates=<MeasurementCoordinates.XYZ: 'XYZ'>, user_defined_cs=None, leader_line_length=None)[source]¶

Creates an angle measurement from one line to another line.

Parameters:
  • line_from (trimatic.Line) – The first line.
  • line_to (trimatic.Line) – The second line.
  • measurement_coordinates (trimatic.MeasurementCoordinates, optional) – Choose to measure the angle in all 3 axes or only one plane.
  • user_defined_cs (trimatic.CoordinateSystem, optional) – Single entity of trimatic.CoordinateSystem. If this parameter is not valid, then the world coordinate system will be applied. The measurement will be taken based on the coordinate system of the selected entity.
  • leader_line_length (float, optional) – The length of the leader lines. When None, the length is computed automatically.
Returns:

Created measurement.
Return type:

trimatic.AngleMeasurement
Raises:

RuntimeError, ValueError
Example: 
import trimatic

p1 = (0,0,0)
p2 = (21,4.3,67)
line1 = trimatic.create_line(p1,p2)
line2 = trimatic.create_line((0,0,0),(-3,-456,45))
trimatic.create_angle_measurement_line_to_line(line1,line2,leader_line_length=300)

create_distance_measurement(point1, point2, measurement_coordinates=<MeasurementCoordinates.XYZ: 'XYZ'>, user_defined_cs=None)[source]¶

Measures the distance between 2 points.

Parameters:
  • point1 ([float] | (float) | trimatic.Point) – Creates first point.
  • point2 ([float] | (float) | trimatic.Point) – Creates second point.
  • measurement_coordinates (trimatic.MeasurementCoordinates, optional) – Specify coordinates to measure the angle in all 3 axes or only one plane.
  • user_defined_cs (trimatic.CoordinateSystem, optional) – Single entity of trimatic.CoordinateSystem. When this parameter is None, the world coordinate system will be applied. The measurement will be taken based on the coordinate system of the selected entity.
Returns:

Created measurement.
Return type:

trimatic.DistanceMeasurement
Raises:

RuntimeError, ValueError
Example: 
import trimatic
import os

trimatic.open_project(os.path.dirname(trimatic.get_application_path()) + r"\DemoFiles\Aorta.mxp")
aorta = trimatic.find_part("Aorta")

p1 = (0,0,0)
p2 = (21,4.3,67)
trimatic.create_distance_measurement(p1,p2,user_defined_cs=aorta.object_coordinate_system)
