sketch¶

create_sketch(planes)[source]¶

Creates a sketch from one or more existing plane(s).

Parameters:planes (trimatic.Plane | [trimatic.Plane]) – Planes that define the sketch. When multiple planes are specified, they are averaged.
Raises:ValueError, RuntimeError
Returns:sketch
Return type:trimatic.Sketch
Example: 
import trimatic

tmp_plane_for_sketch = trimatic.create_plane_3_points(point1=(0,0,0), point2=(10,10,10), point3=(0,20,10))
sketch1 = trimatic.create_sketch(planes=tmp_plane_for_sketch)
trimatic.delete(tmp_plane_for_sketch)
import_intersection(sketch, entities, construction=False)[source]¶

Imports entities into a sketch by making intersections.

Parameters:
  • entities (trimatic.Curve | trimatic.CurveSet | trimatic.Surface | trimatic.SurfaceSet | trimatic.Part | trimatic.Plane | trimatic.Arc | trimatic.Line | trimatic.Point | trimatic.SurfaceBorder | trimatic.SurfaceContour | [trimatic.Curve | trimatic.CurveSet | trimatic.Surface | trimatic.SurfaceSet | trimatic.Part | trimatic.Plane | trimatic.Arc | trimatic.Line | trimatic.Point | trimatic.SurfaceBorder | trimatic.SurfaceContour]) – One or more entities to intersect with.
  • sketch (trimatic.Sketch) – Target sketch for import.
  • construction (bool, optional) – When True, geometry is imported as construction.
Raises:

ValueError, RuntimeError
Returns:

sketch
Return type:

trimatic.Sketch
Example: 
import trimatic

cylinder = trimatic.create_cylinder_part(point_1=(0,0,0), point_2=(0,10,0), radius=5)
sketch1 = trimatic.create_sketch(planes=cylinder.object_coordinate_system.zx_plane.convert_to_plane())
trimatic.import_intersection(sketch=sketch1, entities=cylinder, construction=True)
import_outline(sketch, entities, construction=False)[source]¶

Imports outline of entities into a sketch.

Parameters:
  • entities (trimatic.Surface | trimatic.SurfaceSet | trimatic.Part | [trimatic.Surface | trimatic.SurfaceSet | trimatic.Part]) – One or more entities to compute outline.
  • sketch (trimatic.Sketch) – Target sketch for import.
  • construction (bool, optional) – When True, geometry is imported as construction.
Raises:

ValueError, RuntimeError
Returns:

sketch
Return type:

trimatic.Sketch
Example: 
import trimatic

box = trimatic.create_box_part(corner_point=(-5,-5,-5), x_extent=10, y_extent=10, z_extent=10)
sketch = trimatic.create_sketch(planes=box.object_coordinate_system.xy_plane.convert_to_plane())
trimatic.translate(sketch, (0,0,10))
surface_front = box.find_surface("Front")

trimatic.import_outline(sketch=sketch, entities=surface_front, construction=True)
import_projection(sketch, entities, construction=False)[source]¶

Projects entities onto a sketch.

Parameters:
  • entities (trimatic.Curve | trimatic.CurveSet | trimatic.Arc | trimatic.Line | trimatic.Point | trimatic.SurfaceBorder | trimatic.SurfaceContour | [trimatic.Curve | trimatic.CurveSet | trimatic.Arc | trimatic.Line | trimatic.Point | trimatic.SurfaceBorder | trimatic.SurfaceContour]) – One or more entities to project.
  • sketch (trimatic.Sketch) – Sketch to project onto.
  • construction (bool, optional) – When True, geometry is imported as construction.
Raises:

ValueError, RuntimeError
Returns:

sketch
Return type:

trimatic.Sketch
Example: 
import trimatic

box = trimatic.create_box_part(corner_point=(-5, -5,-5), x_extent=10, y_extent=10, z_extent=10)
sketch1 = trimatic.create_sketch(planes=box.object_coordinate_system.xy_plane.convert_to_plane())
surface_left = box.find_surface("Left")
border_left = surface_left.get_border()
sketch1 = trimatic.import_projection(sketch=sketch1, entities=[border_left], construction=True)
sketch_to_curves(sketch, add_to=None)[source]¶

Converts a sketch to curves.

Parameters:
  • sketch (trimatic.Sketch) – Sketch to export.
  • add_to (trimatic.Part, optional) – Target part to add the resulting curve to. If set to None, a new part is created.
Raises:

ValueError, RuntimeError
Returns:

Part that contains the output curves.
Return type:

trimatic.Part
Example: 
import trimatic

prim1=trimatic.create_cylinder_part(point_1=(0,0,0), point_2=(0,10,0), radius=5)
plane1 = trimatic.create_plane_3_points(point1=(0,0,0), point2=(10,12,13), point3=(0,25,12))
sketch1 = trimatic.create_sketch(planes=plane1)
trimatic.import_intersection(sketch1, prim1, construction=False)
curve1=trimatic.sketch.sketch_to_curves(sketch1)
trimatic.delete (prim1)
trimatic.delete (plane1)
trimatic.delete (sketch1)
