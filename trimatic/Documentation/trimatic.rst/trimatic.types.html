__module__ = 'trimatic.types'¶
add_items(entities)[source]¶

Add items to group.

Parameters:entities ((trimatic.Object)) – A tuple of objects to add in the group.
Raises:RuntimeError
Example: 
import trimatic

# create parent group
parent_group = trimatic.create_group("Parent Group")
        
# create box
box1 = trimatic.create_box_part((0,0,10), 5, 5, 5)
        
# create child group with box
child_group = trimatic.create_group("Child Group")

# add box in child_group
child_group.add_items(entities=box1)

# add child group as subgroup in parent group
parent_group.add_items(entities=child_group)
