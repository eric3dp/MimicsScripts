get_view()[source]¶

Returns the view direction from the 3D view.

Returns:Tuple of 2 vectors. The view vector and view up vector.
Return type:((float, float, float), (float, float, float))
Raises:RuntimeError
Example:

view=trimatic.get_view()
trimatic.view_custom(view_vector=view[0],up_vector=view[1])
show_only(entities)[source]¶

Zoom and show only the specified entities and hide the non-specified entities.

Entities refer to all root level entities including groups and subgroups.

Parameters:entities (trimatic.Object | trimatic.Group | [trimatic.Object | trimatic.Group]) – Either a single or list of trimatic.Object and/or trimatic.Group
Returns:Operation successful or not.
Return type:bool
Raises:ValueError, RuntimeError
Example: 
import trimatic 
import os 
trimatic.open_project(os.path.split(trimatic.get_application_path())[0] + r"\DemoFiles\BonePlate.mxp")
plate= trimatic.find_part('Plate')
screws=trimatic.find_part('Screws')
trimatic.show_only(entities=(plate,screws))
view_custom(view_vector, up_vector)[source]¶

Sets the 3D view to a custom view direction. View is zoomed such that all objects are visible (“Unzoom All”).

Parameters:
  • view_vector (trimatic.Point) – view_vector is an eye vector, this vector is orthogonal to the screen and directed from the user. If passed vectors are not orthogonal, up_vector is set as orthogonal to view_vector in the plane determined by up and view vectors.
  • up_vector (trimatic.Point) – up_vector is aligned with Y-axis (bottom to top) of the window.
Returns:

Operation successfull or not.
Return type:

bool
Raises:

RuntimeError, ValueError
Example: 
import trimatic

view=trimatic.get_view()
trimatic.view_custom(view_vector=view[0],up_vector=view[1])
view_default(view)[source]¶

Sets the 3D view to a predefined view direction. View is zoomed such that all objects are visible (“Unzoom All”).

Top: Direction of visualization is perpendicular to the XY plane, in the Z-direction.

Bottom: Direction of visualization is perpendicular to the XY plane, against the Z- direction.

Left: Direction of visualization is perpendicular to the YZ plane, in the X-direction.

Right: Direction of visualization is perpendicular to the YZ plane, against the X-direction.

Front: Direction of visualization is perpendicular to the XZ plane, in the Y-direction.

Back: Direction of visualization is perpendicular to the XZ plane, against the Y-direction.

Isometric: Direction of visualization is (1,1, -1).

Parameters:view (trimatic.DefaultView) – Which view to set.
Returns:Operation successfull or not.
Return type:bool
Raises:RuntimeError, ValueError
Example: 
import trimatic

trimatic.view_default(view=trimatic.DefaultViews.Left)
zoom(entities)[source]¶

Zoom the view in such a way that the specified entities will be enclosed in the view. The resulting view will be the same as when using “Zoom Selection” in GUI for the specified entities.

Parameters:entities (trimatic.Object | [trimatic.Object]) – Either a single object or list of trimatic.Objects
Returns:Operation successful or not.
Return type:bool
Raises:RuntimeError, ValueError
Example: 
import trimatic
import os

trimatic.open_project(os.path.dirname(trimatic.get_application_path()) + r"\DemoFiles\Aorta.mxp")

aorta = trimatic.find_part("Aorta")
some_surfaces = aorta.find_surface_sets("In-outlet")[0].get_surfaces()[0:3]

trimatic.zoom(entities=some_surfaces)
