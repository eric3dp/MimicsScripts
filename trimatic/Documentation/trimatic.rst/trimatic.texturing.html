texturing¶

convert_texture_to_mesh(entities, detail_size)[source]¶

Converts textures to mesh.

Input entities are grouped by the parts they belong to. For every group, a new part will be created. To control texture offsets, use set_texture_properties.

Parameters:
  • entities (trimatic.Part | trimatic.UVMap | trimatic.UVRegion | trimatic.Texture | [trimatic.Part | trimatic.UVRegion | trimatic.UVMap | trimatic.Texture]) – Either a single object or heterogenous list of entities.
  • detail_size (float) – The resolution at which the texture will be processed to generate the mesh.
Returns:

Newly created part(s).
Return type:

(trimatic.Part)
Raises:

ValueError | RuntimeError
Example: 
import trimatic
import os

box = trimatic.create_box_part([0, 0, 0], 10, 10, 10)
top_surface = box.find_surface("Top")
uv_map = trimatic.create_uv_maps(entities = top_surface, preserve_surface_connections = False)
path_to_image = os.path.dirname(trimatic.get_application_path()) + r"\TextureLibrary\008_honeycomb_black.png"
texture = trimatic.create_uv_based_texture(uv_maps = uv_map, image_path = path_to_image, tile_image = True)
trimatic.set_texture_properties(entities = texture, white_offset = 3.0, black_offset = 0, accuracy = 0.3)
new_part = trimatic.convert_texture_to_mesh(entities = texture, detail_size = 1)
create_fade_texture(*args, texture, fading_guides, is_black=True, influence_distance=1.0, fading_factor=1.0)[source]¶

Fades the texture images to the defined color and distance based on the input guiding curves.

Parameters:
  • texture (trimatic.Texture) – A single texture to be applied with fading.
  • fading_guides (trimatic.Curve | trimatic.SurfaceContour | trimatic.SurfaceBorder | [trimatic.Curve | trimatic.SurfaceContour | trimatic.SurfaceBorder]) – One or multiple guide curves for the fading lines. Heterogeneous list of entities is allowed.
  • is_black (bool, optional) – The color that the fading operation should fade to. If True, the fade color will be set to black, else will be white.
  • influence_distance (float, optional) – The distance of the regions influenced by the fading operation in XYZ coordinate system.
  • fading_factor (float, optional) – The intensity of the fading at the position of the fading guide.
Returns:

One or more faded textures.
Return type:

trimatic.Texture | (trimatic.Texture)
Raises:

ValueError | RuntimeError
Example: 
import trimatic
import os

trimatic.open_project(os.path.dirname(trimatic.get_application_path()) + r"\DemoFiles\FadeTexture.mxp")
part = trimatic.find_part("Box-001")
texture = part.find_texture("006_honeycomb round.png")
front_border = part.find_surface('Front').get_border()
uv_regions = trimatic.create_fade_texture(texture=texture, fading_guides=[front_border], is_black=True, influence_distance=2.0, fading_factor=0.5)
create_procedural_leather_texture(*args, entities, texture_resolution=1000, primary_cell_size=1.0, fading_factor=0.15, is_apply_spots=False, is_round_cells=False, secondary_cell_size_factor=1.0, secondary_cell_intensity=1.0)[source]¶

Create and apply leather texture procedurally to the desired UV Map(s).

Parameters:
  • entities (trimatic.Part | trimatic.UVMap | [trimatic.Part | trimatic.UVMap]) – Part(s) with UV Map(s) to be applied with a leather texture.
  • texture_resolution ((int)) – The resolution of the procedural texture bitmap.
  • primary_cell_size ((float)) – The main voronoi cell sizes.
  • fading_factor ((float)) – The fading factor of the voronoi cells’ lines to the center of cell. Useful for generating different kinds of bump profiles for the voronoi cells.
  • is_apply_spots ((bool)) – If True, adds spots to the leather texture.
  • is_round_cells ((bool)) – If True, voronoi cell lines will be smoothed to create more organic cells.
  • secondary_cell_size_factor ((float)) – The factor of the secondary voronoi cells relative to the primary voronoi cells. When set to 1, no secondary cells will be generated.
  • secondary_cell_intensity ((float)) – The intensity of the secondary voronoi cell pixels in the final image texture.
Returns:

A tuple of Texture(s) generated by the operation.
Return type:

(trimatic.Texture)
Raises:

ValueError | RuntimeError
Example: 
import trimatic

box = trimatic.create_box_part([0, 0, 0], 10, 10, 10)
target_surface = box.find_surface("Top")
uv_map = trimatic.create_uv_maps(entities = target_surface, preserve_surface_connections = False)
texture = trimatic.create_procedural_leather_texture(entities=uv_map, 
                                                     primary_cell_size=1,
                                                     texture_resolution=100,
                                                     fading_factor=0.15,
                                                     secondary_cell_size_factor=1,
                                                     secondary_cell_intensity=1)
create_uv_based_rectangular_pattern_with_deformation(*args, uv_map, pattern_entity, margin_u=0.2, margin_v=0.2, preserve_pattern_height=False, height_offset=0.0, preserve_boundary_entity=True)[source]¶

Patterns the entity across the UV Map with deformation in the output.

Parameters:
  • uv_map (trimatic.UVMap) – Parameterized entity to be used as the patterning guide.
  • pattern_entity (trimatic.Part) – One part to be patterned across the parameterized surface.
  • margin_u (float, optional) – The distance between the pattern entities in the U direction in UV units.
  • margin_v (float, optional) – The distance between the pattern entities in the U direction in UV units.
  • preserve_pattern_height (bool, optional) – If True, preserve the original pattern entity height in the patterned output.
  • height_offset (float, optional) – The distance between the bottom of the pattern entity and the surface with the UV Map on.
  • preserve_boundary_entity (bool, optional) – If True, all the partial pattern entities are preserved in the final output.
Returns:

One part with the output of the patterning operation.
Return type:

trimatic.Part
Raises:

ValueError | RuntimeError
Example: 
import trimatic
import os

trimatic.open_project(os.path.dirname(trimatic.get_application_path()) + r"\DemoFiles\Cup.mxp")
target_part = trimatic.find_part("Lattice Volume")
target_surface = target_part.find_surface_set("Subtraction result-001-001+-001-001-2+")
trimatic.create_uv_maps(entities = target_surface, preserve_surface_connections = True)
uv_map = target_part.find_uv_map('UV Map-001')
pattern_entity_part = trimatic.create_cylinder_part((0,0,0), (0,0,2), 1)

uv_based_rectangular_pattern_part = trimatic.create_uv_based_rectangular_pattern_with_deformation(uv_map = uv_map, pattern_entity = pattern_entity_part,
                                                                                                  margin_u = 0.1, margin_v = 0.1, height_offset = 1.0)

create_uv_based_texture(uv_maps, image_path, tile_image=True)[source]¶

Create UV Based Texture on the selected UV Maps.

Parameters:
  • uv_maps (trimatic.UVMap | [trimatic.UVMap]) – Parameterized entities to be textured.
  • image_path (string) – Path to the texture image file in the user’s computer. Supported image formats are: .bmp, .jpeg, .gif, .png, .tiff.
  • tile_image (bool, optional) – If True, the tile will be repeated in all the UV boxes in the UV Map.
Returns:

Newly created textures in the selected UV Maps.
Return type:

trimatic.Texture | (trimatic.Texture)
Raises:

ValueError | RuntimeError
Example: 
import trimatic
import os

box = trimatic.create_box_part([0, 0, 0], 10, 10, 10)
target_surface = box.find_surface("Top")
uv_map = trimatic.create_uv_maps(entities = target_surface, preserve_surface_connections = False)
path_to_image = os.path.dirname(trimatic.get_application_path()) + r"\TextureLibrary\008_honeycomb_black.png"
texture = trimatic.create_uv_based_texture(uv_maps=uv_map, image_path=path_to_image,tile_image=True)

set_texture_properties(entities, white_offset, black_offset, accuracy)[source]¶

Sets the texture properties.

Parameters:
  • entities (trimatic.UVMap | trimatic.Texture | [trimatic.UVMap | trimatic.Texture]) – Either a single object or a heterogeneous list of entities.
  • white_offset (float) – The white offset value for 3D texture.
  • black_offset (float) – The black offset value for 3D Texture.
  • accuracy (float) – The accuracy for the slice-based texture operations.
Returns:

Textures with properties edited.
Return type:

trimatic.Texture | (trimatic.Texture)
Raises:

ValueError
Example: 
import trimatic
import os

box = trimatic.create_box_part((0.0,0.0,0.0), 6.0, 8.0, 7.0)
uv_map = trimatic.create_uv_maps(entities=(box.find_surface('Right'), box.find_surface('Front')), single_uv_map=True)
path_to_image = os.path.dirname(trimatic.get_application_path()) + r"\TextureLibrary\008_honeycomb_black.png"
texture = trimatic.create_uv_based_texture(uv_maps=uv_map, image_path=path_to_image, tile_image=True)

textures = trimatic.set_texture_properties(entities=uv_map, white_offset=3.0, black_offset=4.0, accuracy=0.3)
