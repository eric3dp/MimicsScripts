attach_curve(entities, target_entities=None, distance_threshold=5.0)[source]¶

Attaches curve(s) to the specified part(s)/surface(s). The curve is first attracted and then attached.

Parameters:
  • entities (trimatic.Curve | trimatic.CurveSet | trimatic.Sketch | [trimatic.Curve | trimatic.CurveSet | trimatic.Sketch] | (trimatic.Curve | trimatic.CurveSet | trimatic.Sketch)) – Either single object or list of curves, curve sets or sketches.
  • target_entities (trimatic.Part | trimatic.Surface | [trimatic.Part | trimatic.Surface], optional) – The target entities that the curves will be attached to. If set to None, the curves will be attached to the part they currently belong to.
  • distance_threshold (float, optional) – Curve segments that are within the distance threshold will be attached.
Returns:

Attached curve(s).
Return type:

trimatic.Curve | (trimatic.Curve)
Raises:

RuntimeError, ValueError
Example: 
import trimatic
import os

trimatic.open_project(os.path.dirname(trimatic.get_application_path()) + r"\DemoFiles\Aorta.mxp")
p = trimatic.find_part("Aorta")
c = trimatic.create_curve([(175.773, -154.0134, 45.9217), (184.0625, -127.4913, 43.0846)], p)
attracted_c = trimatic.attach_curve(c, p, 15)
attract_curve(entities, target_entities=None, distance_threshold=5.0, attach_curve=True)[source]¶

Attracts curve(s) to the specified part(s)/surface(s).

If multiple curves are specified as input, operation will succeed if at least one of the curves is successfully attracted.

Parameters:
  • entities (trimatic.Curve | trimatic.CurveSet | trimatic.Sketch | [trimatic.Curve | trimatic.CurveSet | trimatic.Sketch] | (trimatic.Curve | trimatic.CurveSet | trimatic.Sketch)) – Either single object or list of curves, curve sets or sketches.
  • target_entities (trimatic.Part | trimatic.Surface | [trimatic.Part | trimatic.Surface], optional) – The target entities that the curves will be attracted to. If set to None, the curves will be attracted to the part they currently belong to.
  • distance_threshold (float, optional) – Curve segments that are within the distance threshold will be attracted.
  • attach_curve (bool, optional) – If True, the target_entities will be retriangulated and the curve will be attached to the part.
Returns:

Attracted curve(s).
Return type:

trimatic.Curve | (trimatic.Curve)
Raises:

RuntimeError, ValueError
Example: 
import trimatic
import os

trimatic.open_project(os.path.dirname(trimatic.get_application_path()) + r"\DemoFiles\Aorta.mxp")
p = trimatic.find_part("Aorta")
c = trimatic.create_curve([(175.773, -154.0134, 45.9217),(184.0625, -127.4913, 43.0846)], p)
attracted_c = trimatic.attract_curve(c, p, 3.0, True)
connect_curves(entities, distance_threshold=0.1)[source]¶

Connects curves together.If there is a small overlap, this will be removed.

If there is a small gap between the endings of two selected curves, they will be connected with a free curve, if the gap is smaller than a specified distance.

Parameters:
  • entities ([trimatic.Curve]) – List of curves to connect.
  • distance_threshold (float, optional) – Curve endings that are within the distance threshold will be connected and merged.
Returns:

Connected curve.
Return type:

trimatic.Curve
Raises:

RuntimeError, ValueError
Example: 
import trimatic
import os

trimatic.open_project(os.path.dirname(trimatic.get_application_path()) + r"\DemoFiles\Aorta.mxp")
p = trimatic.find_part("Aorta")
c1 = trimatic.create_curve([(175.773, -154.0134, 45.9217),(184.0625, -127.4913, 43.0846)], p)
c2 = trimatic.create_curve([(199, -134, 45),(195, -156, 47)], p)
c = p.get_curves()
unattached_c = trimatic.connect_curves([c[0],c[1]],60)
attached_c = trimatic.attract_curve(unattached_c, p, 15)

create_geometry_curve_curvature_based(entities, curvature_type=<CurvatureType.maximal: 'Maximal'>, radius_threshold=16.0, sharp_edge_threshold=30.0)[source]¶

Creates curves where the surface has a curvature specified by the input parameters.

Parameters:
  • entities (trimatic.Part | trimatic.Surface | trimatic.SurfaceSet | [trimatic.Part | trimatic.Surface | trimatic.SurfaceSet]) – Either single object or list of parts, surfaces or surface sets.
  • curvature_type (trimatic.CurvatureType, optional) – Method used to calculate curvature information.
  • radius_threshold (float, optional) – Threshold of the average triangle curvature. A curve is created on the common edge of two triangles when one triangle has a curvature smaller than the threshold and the other triangle has a curvature bigger than the threshold.
  • sharp_edge_threshold (float, optional) – If the angle between two triangles is smaller than the defined threshold, the common edge is considered sharp. Whether a curve is created on the comon triangle edge depends on the radius threshold.
Returns:

The created geometry curves.
Return type:

trimatic.Curve | (trimatic.Curve)
Raises:

RuntimeError, ValueError
Example: 
import trimatic

cone_part = trimatic.create_cone_part(origin=(0,0,0), direction=(0,0,1), height=10, bottom_radius=5)
trimatic.create_geometry_curve_curvature_based(cone_part, trimatic.CurvatureType.maximal, 16, 30)

Projects curve(s) to the specified part(s)/surface(s).

  • target_entities (trimatic.Part | trimatic.Surface | [trimatic.Part | trimatic.Surface]) – The target entities that the curves will be projected to.
  • visibility_threshold (float | None, optional) – If None, projected curve segments are connected along the part of the mesh in the projection direction even if the region of the mesh is not visible from the original curve. Projected curve(s).
