cadlink_quad_dominant(*args, entity, format=<CADLinkFormat.step: 'step'>, output_directory, nurbs_only=False, border_tolerance=None, analytical_structures=<CADLinkAnalyticalStructures.slit: 2>, preserve_surface_structure=True, improve_mesh=True, patch_size=None, number_of_control_points=None, smoothing_strength=0.0)[source]¶

The CAD link module makes it possible to convert a triangulated mesh to an Iges or Step file using Quad dominant method.

Parameters:
  • entity (trimatic.Part) – part to convert.
  • format (trimatic.CADLinkFormat, optional) – CAD file format, igs or step.
  • output_directory (string) – Destination path to save the CAD file.
  • nurbs_only (bool, optional) – When True, all surfaces will be converted to NURBS surfaces. When False, the opertion will first search for a match of the triangulated surfaces with an analytical primitive like a plane, cylinder etc.
  • border_tolerance (float, optional) – The border tolerance is the tolerance with which the file is read. When two points are further apart than this tolerance, they are considered to be 2 different points. When they are closer to each other as the defined tolerance, they are considered to fall together. When None is specified, tolerance will be estimated automatically (Like “Estimate” button in GUI).
  • analytical_structures (trimatic.CADLinkAnalyticalStructures, optional) – Trimming loop representation for closed analytical surfaces.
  • preserve_surface_structure (bool, optional) – Specifies whether the surface structure should be preserved or not.
  • improve_mesh (bool, optional) – Specifies whether or not the mesh improving shall be performed before reverse (recommended).
  • patch_size (float, optional) – Desired edge length for decomposition via quadrangulation algorithm.
  • number_of_control_points (integer, optional) – Number of control points. If None, then estimate automatically.
  • smoothing_strength (float, optional) – Strength of NURBS surfaces smoothing. Value from 0 to 1.
Returns:

Names of written file.
Return type:

string
Raises:

RuntimeError, ValueError
Example: 
import trimatic

cylinder = trimatic.create_cylinder_part((0,0,0), (10,0,0), 5)
trimatic.cadlink_quad_dominant(entity=cylinder, output_directory="C:/MedData")

The CAD link module makes it possible to convert a triangulated mesh to an Iges or Step file using Voronoi based patching method.

Parameters:
  • entity (trimatic.Part) – part to convert.
  • format (trimatic.CADLinkFormat, optional) – CAD file format, igs or step.
  • output_directory (string) – Destination path to save the CAD file.
  • nurbs_only (bool, optional) – When True, all surfaces will be converted to NURBS surfaces. When False, the opertion will first search for a match of the triangulated surfaces with an analytical primitive like a plane, cylinder etc.
  • border_tolerance (float, optional) – The border tolerance is the tolerance with which the file is read. When two points are further apart than this tolerance, they are considered to be 2 different points. When they are closer to each other as the defined tolerance, they are considered to fall together. When None is specified, tolerance will be estimated automatically (Like “Estimate” button in GUI).
  • analytical_structures (trimatic.CADLinkAnalyticalStructures, optional) – Trimming loop representation for closed analytical surfaces.
  • preserve_surface_structure (bool, optional) – Specifies whether the surface structure should be preserved or not.
  • improve_mesh (bool, optional) – Specifies whether or not the mesh improving shall be performed before reverse (recommended).
  • patch_density (trimatic.CADLinkPatchDecompositionType, optional) – Recommended density of resulting parametric faces. Used with regular decomposition method only. It is called ‘recommended density’ because it can contradict the reversing tolerance parameter that is considered as a superior one. The default ‘adaptive’ value is used to vary patch density depending on mesh curvature. Other possible value can be None, Low, Below normal, Normal, Above normal, or High. If None, reverse must be performed as a single patch in this case.
  • number_of_control_points (int, optional) – Number of control points. If None, then estimate automatically.
  • smoothing_strength (float, optional) – Strength of NURBS surfaces smoothing. Value from 0 to 1.
Returns:

Names of written file.
Return type:

string
Raises:

RuntimeError, ValueError
Example: 
import trimatic

cylinder = trimatic.create_cylinder_part((0,0,0), (10,0,0), 5)
trimatic.cadlink_voronoi_based(entity=cylinder, output_directory="C:/MedData")
