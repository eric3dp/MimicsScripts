arc_to_arc_align(fixed_entity, moving_entity, move_along_entities=(), coincident=True)[source]¶

Aligns two (or more) entities, through arc-fitting method, based on the center and normal orientation of the selected arcs.

Parameters:
  • fixed_entity (trimatic.Curve | trimatic.Surface | trimatic.Arc) – Reference entity to be aligned with.
  • moving_entity (trimatic.Curve | trimatic.Surface | trimatic.Arc) – Entity to align/move. For curves or surfaces, the part they belong to will be moved.
  • move_along_entities (trimatic.Object | [trimatic.Object] | (trimatic.Object), optional) – Either single object, empty or heterogeneous list/tuple of root level entities (parts, analytical primitives) that will be aligned along with the moving_entity.
  • coincident (bool, optional) – If True, centre points of the input arcs will be made coincident and their normals will be parallel.
Returns:

Aligned entities. This will be a combination of moving_entity and move_along_entities.
Return type:

trimatic.Object | (trimatic.Object)
Raises:

RuntimeError, ValueError
Example: 
import trimatic
import os

app_folder = os.path.dirname(trimatic.get_application_path())
trimatic.open_project(app_folder + r"\DemoFiles\Aorta.mxp")
trimatic.import_project(app_folder + r"\DemoFiles\Flange.mxp")

aorta = trimatic.find_part("Aorta")
inlet_surface = aorta.find_surface_sets("In-outlet")[0].get_surfaces()[3]

flange = trimatic.find_part("Cylinder-012-2")
flange_curve = flange.get_curves()[0]
flange_arc = trimatic.create_circle_arc_fit(flange_curve)

result = trimatic.arc_to_arc_align(inlet_surface, flange_arc, flange)
global_registration(*args, fixed_entity, moving_entity, move_along_entities=(), distance_threshold=None, number_of_iterations=10, subsample_percentage=15)[source]¶

Performs final registration of entities based on a heuristic algorithm.

Parameters:
  • fixed_entity (trimatic.Part | trimatic.Surface | trimatic.SurfaceSet | trimatic.MarkedTriangles) – Reference entity to be aligned with.
  • moving_entity (trimatic.Part | trimatic.Surface | trimatic.SurfaceSet | trimatic.MarkedTriangles) – Entity to align/move.
  • move_along_entities (trimatic.Object | [trimatic.Object] , optional) – Either single object or heterogeneous list of root level entities (parts, analytical primitives) that will be aligned along with the moving_entity.
  • distance_threshold (float, optional) – If distance between a point on fixed and moving entity is less than the threshold value, the point will be used for registration. Set to None for automatic calculation.
  • number_of_iterations (int, optional) – Increases the number of passes for higher quality but increases computation time.
  • subsample_percentage (int, optional) – Defines how many points (in percentage) will be used for registration.
Returns:

Average distance deviation after registration or None if operation failed.
Return type:

float
Raises:

RuntimeError, ValueError
Example: 
import trimatic
import os

aorta1 = trimatic.import_part_stl(os.path.dirname(trimatic.get_application_path()) + r"\DemoFiles\Aorta1.stl")
aorta2 = trimatic.import_part_stl(os.path.dirname(trimatic.get_application_path()) + r"\DemoFiles\Aorta2.stl")

trimatic.global_registration(fixed_entity=aorta1,moving_entity=aorta2,distance_threshold=300)

This function mirrors the selected part around a mirror plane defined by the user.

  • origin (trimatic.Point) – origin of the mirror plane.
  • normal (trimatic.Point) – normal of the mirror plane.
Returns:

mirrored entities
Return type:

(trimatic.Object)

RuntimeError, ValueError, AttributeException

import trimatic
import os

trimatic.open_project(os.path.dirname(trimatic.get_application_path()) + r"\DemoFiles\Skull.mxp")

normal = sagittal_plane.normal
mirrored_skull = trimatic.mirror(duplicate_skull, origin=origin, normal=normal)

Performs registration based on 2 sets of points, e.g. landmarks

Parameters:
  • fixed_points ([(float,float,float)]) – Reference points to be aligned with.
  • moving_points ([(float,float,float)]) – For every fixed point, a corresponding “moving point”. The points won’t actually move, but indicate points on the moving entities.
  • moving_entities (trimatic.Object | [trimatic.Object]) – Either single object or heterogeneous list of root level entities. (parts, analytical primitives) that will be aligned along with the moving_entity.
Returns:

moved entities
Return type:

(trimatic.Object)
Raises:

RuntimeError, ValueError, AttributeException
Example: 
import trimatic

sphere = trimatic.create_sphere_part([0, 0, 0], 20, tolerance=0.05)
sphere_cog_init = trimatic.compute_center_of_gravity(sphere)
[sphere_points_init, tri] = sphere.get_triangles()
sphere_transformed = trimatic.translate(sphere, number_of_copies=1, translation_vector=[11, 26, 98])
sphere_transformed = trimatic.rotate_around_axes(sphere_transformed, number_of_copies=0, angle_axes=[5, 8, 7], rotation_origin=[5, 9, 36])
cog_sphere_transformed = trimatic.compute_center_of_gravity(sphere_transformed)

scale_delta(entities, delta, center_object=True, number_of_copies=0)[source]¶

Scales entities by the specified distance along each major axis.

Parameters:
  • entities (trimatic.Part | [trimatic.Part] | (trimatic.Part)) – Either single object or heterogeneous list of objects.
  • delta ((float, float, float)) – An (x,y,z) tuple with values representing the scale distance along each major axis.
  • center_object (bool, optional) – position of scale, by default center around object. If False, center at midpoint of all objects.
  • center_object – If True, scaling is done relative to the center of each input entity independently. Otherwise, scaling is done against the average center of all the input entities.
  • number_of_copies (int, optional) – Number of additional copies of the input entities to create. Each copy is scaled by factor relative to the last copy made.
Returns:

Scaled entities (including any copies).
Return type:

trimatic.Part | (trimatic.Part)
Raises:

RuntimeError, ValueError
Example:

Updates the object coordinate system to a defined coordinate system.

  • method (trimatic.UpdateOCSMethod, optional) – Updates the OCS based on one of the methods
