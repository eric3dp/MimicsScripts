subscription¶

obj_created_status()[source]¶

Check subscription status for object created callback notification.

Returns:True if subscribed to callback notification.
Return type:bool
Raises:RuntimeError
Example: 
import trimatic

trimatic.obj_created_status()
obj_created_subscribe(callable_object)[source]¶

Subscribe to object created callback notification.

Parameters:callable_object (Callable) – User defined function
Returns:True if callback notification was successfully subscribed.
Return type:bool
Raises:RuntimeError, ValueError
Example: 
import trimatic

# User defined callback function
def myCallbackFunction(my_obj):
    print(f'Hello, {my_obj}!')

trimatic.obj_created_subscribe(myCallbackFunction)
