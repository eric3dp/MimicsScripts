copy_to_part(entities, destination=None)[source]¶

Copies surfaces, curves, and surface or curve sets from one part to another.

Parameters:
  • entities ([ trimatic.Surface | trimatic.SurfaceSet | trimatic.Curve | trimatic.CurveSet | trimatic.MarkedTriangles ]) – Either single object or heterogeneous list of objects.
  • destination (trimatic.Part) – The part where the data has to be copied to. When None is specified, a new one will be created.
Returns:

copied entities or newly created part when a new part was created.
Return type:

( trimatic.Surface | trimatic.SurfaceSet | trimatic.Curve | trimatic.CurveSet ) | trimatic.Part
Raises:

RuntimeError, ValueError
Example: 
import trimatic

box = trimatic.create_box_part([0, 0, 0], 20, 20, 20)
surface = box.find_surface("Right")
result = trimatic.copy_to_part(surface, None)
delete(entities)[source]¶

Deletes the specified objects.

Deleting an object multiple times or attempting to access a deleted object will raise an AttributeError.

Parameters:entities (trimatic.Object | [trimatic.Object] | (trimatic.Object)) – Either single object or heterogeneous list of parts, surfaces, surface sets, etc.
Returns:True if delete is successful.
Return type:bool
Raises:RuntimeError, AttributeError
Example: 
import trimatic

box = trimatic.create_box_part([0, 0, 0], 10, 10, 10)
box_duplicate = trimatic.duplicate(box)

trimatic.delete(box)

find_objects_with_metadata(name, value)[source]¶

Finds objects whose metadata matches specified criteria. Objects without metadata will be ignored.

Parameters:
  • name (string) – Name to find. When None is specified, name can be anything.
  • value (string) – Value to find. When None is specified, value can be anything.
Returns:

The object(s) matching the search criteria.
Return type:

(trimatic.Object)
Example: 
import trimatic

box = trimatic.create_box_part(corner_point=(0, 0, 0), x_extent=5, y_extent=5, z_extent=5)
box_metadata = box.get_metadata()
box_metadata.create('Name', 'Test')
box_metadata.create('Age', '19')
found_objects = trimatic.find_objects_with_metadata('Name', 'Test')
print(found_objects)

find_part_byguid(guid)[source]¶

Finds a Part object with a specific Globally Unique Identifier (GUID).

This is the recommended method to search for Parts reliably. Names are not guaranteed to be unique, but every Part with the same name will have a different GUID. This is especially useful when exchanging data between 3-matic and Mimics.

Parameters:guid (string) – GUID to find.
Returns:The Part with a matching GUID. None if no match is found.

merge(entities)[source]¶

Merges the specified entities. You can only merge entities of the same type: surfaces with surfaces, parts with parts or graphs with graphs. You can pass either a single entity or multiple entities to merge. For example, you can (i) merge a single SurfaceSet which merges all surfaces in the SurfaceSet, or (ii) merge multiple SurfaceSets, which merges the SurfaceSets into one.

Parameters:entities (trimatic.Part | trimatic.Surface | trimatic.SurfaceSet | trimatic.Graph | trimatic.GraphSet | trimatic.UVRegion | [trimatic.Part | trimatic.Surface | trimatic.SurfaceSet | trimatic.Graph | trimatic.GraphSet | trimatic.UVRegion]) – Either single object or heterogeneous list of objects.
Returns:merged entities.
Return type:trimatic.Part | trimatic.Surface | trimatic.SurfaceSet | trimatic.Graph | trimatic.GraphSet | trimatic.UVRegion
Raises:RuntimeError
Example: 
import trimatic

box1 = trimatic.create_box_part([0, 0, 0], 20, 20, 20)
box2 = trimatic.create_box_part([20, 0, 0], 20, 20, 20)
result = trimatic.merge([box1, box2])
